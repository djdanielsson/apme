## Deprecated with_* loops (M009)

All `with_*` style loops (`with_items`, `with_dict`, `with_fileglob`, etc.) are deprecated in ansible-core 2.19+ in favor of `loop:` combined with appropriate filters.

### Example: violation

```yaml
- name: Install packages
  ansible.builtin.yum:
    name: "{{ item }}"
    state: present
  with_items:
    - httpd
    - nginx
```

### Example: pass

```yaml
- name: Install packages
  ansible.builtin.yum:
    name: "{{ item }}"
    state: present
  loop:
    - httpd
    - nginx
```
