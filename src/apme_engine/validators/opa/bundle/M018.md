## paramiko_ssh connection plugin (M018)

paramiko_ssh connection plugin is deprecated; use ssh connection instead
(removed in 2.21).

**Removal version**: 2.21
**Fix tier**: 1
**Audience**: content

### Detection

Detects `connection: paramiko_ssh` on plays and `ansible_connection: paramiko_ssh`
in task vars. The OPA serializer currently only passes `become`/`become_user`
for play-level options and does not include `vars` in the task options whitelist,
so these examples cannot be tested through the integration harness. OPA unit
tests in `M018_test.rego` validate the Rego logic directly.

### Example: violation

```yaml
- hosts: all
  connection: paramiko_ssh
  tasks:
    - name: Gather facts
      ansible.builtin.setup:
```

### Example: pass

```yaml
- hosts: all
  connection: ssh
  tasks:
    - name: Gather facts
      ansible.builtin.setup:
```

### Remediation

Direct substitution: replace `paramiko_ssh` with `ssh`.
