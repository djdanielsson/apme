## Third-party strategy plugin (M025)

Third-party strategy plugins are deprecated; only `ansible.builtin` strategies
are supported (2.23).

**Removal version**: 2.23
**Fix tier**: 3
**Audience**: content

### Detection

Detects `strategy:` on plays that is not a built-in strategy (`linear`, `free`,
`debug`, `host_pinned`) or `ansible.builtin.*`. The graph-based OPA serializer
now passes `strategy` (and other play-level options) to OPA. OPA unit tests
in `M025_test.rego` validate the Rego logic directly.

### Example: violation

```yaml
- hosts: all
  strategy: mitogen_linear
  tasks:
    - name: Install nginx
      ansible.builtin.apt:
        name: nginx
        state: present
```

### Example: pass

```yaml
- hosts: all
  strategy: linear
  tasks:
    - name: Install nginx
      ansible.builtin.apt:
        name: nginx
        state: present
```

### Remediation

Informational warning only — migrate to a built-in strategy plugin.
