## Bare include removed (M008)

The bare `include:` directive (deprecated since ansible-core 2.4) is removed in 2.19+. Use `include_tasks:` (dynamic) or `import_tasks:` (static) instead.

### Example: violation

```yaml
- include: tasks/setup.yml
```

### Example: pass

```yaml
- ansible.builtin.include_tasks: tasks/setup.yml
```
