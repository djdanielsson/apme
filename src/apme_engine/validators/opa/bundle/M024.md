## include_vars ignore_files as string (M024)

include_vars ignore_files must be a list, not a string (2.24)

**Removal version**: 2.24
**Fix tier**: 1
**Audience**: content

### Detection

Detect ignore_files: with string value in include_vars tasks

### Example: violation

```yaml
- name: Include vars
  ansible.builtin.include_vars:
    dir: vars/
    ignore_files: "*.bak"
```

### Example: pass

```yaml
- name: Include vars
  ansible.builtin.include_vars:
    dir: vars/
    ignore_files:
      - "*.bak"
```

### Remediation

Wrap string value in a YAML list
