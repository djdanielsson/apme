## Removed module (M004)

Detects modules that have been tombstoned (removed) in ansible-core or a collection. When `find_plugin_with_context()` raises `AnsiblePluginRemovedError`, this rule reports the removal message so users know the module is no longer available.

### Example: violation

```yaml
- name: Example play
  hosts: localhost
  connection: local
  tasks:
    - name: Uses removed module
      ansible.builtin.accelerate:
        port: 5099
```

### Example: pass

```yaml
- name: Example play
  hosts: localhost
  connection: local
  tasks:
    - name: Uses valid module
      ansible.builtin.debug:
        msg: "Hello"
```
