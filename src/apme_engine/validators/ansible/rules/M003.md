## Module redirect (M003)

Detects when a module name has been redirected through one or more hops in ansible-core's or a collection's `meta/runtime.yml`. Reports the redirect chain so users can update to the final canonical name.

### Example: violation

```yaml
- name: Example play
  hosts: localhost
  connection: local
  tasks:
    - name: Uses redirected module
      ansible.builtin.command:
        cmd: echo hello
```

### Example: pass

```yaml
- name: Example play
  hosts: localhost
  connection: local
  tasks:
    - name: Uses canonical name
      ansible.builtin.command:
        cmd: echo hello
```
