## Deprecated module (M002)

Uses `find_plugin_with_context()` with `ignore_deprecated=False` to detect modules that have been marked as deprecated in ansible-core's `ansible_builtin_runtime.yml` or a collection's `meta/runtime.yml`. Reports the deprecation warning so users can migrate before the module is removed.

### Example: violation

```yaml
- name: Example play
  hosts: localhost
  connection: local
  tasks:
    - name: Uses deprecated module
      ansible.builtin.include:
        file: other.yml
```

### Example: pass

```yaml
- name: Example play
  hosts: localhost
  connection: local
  tasks:
    - name: Uses current module
      ansible.builtin.include_tasks:
        file: other.yml
```
