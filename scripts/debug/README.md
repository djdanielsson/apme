# Debug utilities

Ad-hoc scripts for local troubleshooting. Not invoked by tox or CI.

See [DEVELOPMENT.md](../../docs/guides/DEVELOPMENT.md#debug-utilities) for usage examples.

| Script | Purpose |
|--------|---------|
| `free-apme-ports.sh` | List or kill processes on APME service ports |
| `ai-prompt-test.py` | Test Abbenay prompt/response for a sample task node |
| `ws-diag.py` | Gateway WebSocket diagnostic client (requires `websockets`) |
